## Mysql Lock 
 
+ **MySQL锁**   
    + S锁(shared(S) locks)，称为共享锁，事务读取记录的时候获取S锁，运行多个事务获取S锁，相互之间不会影响。   
    + X锁(exclusive(X) locks)，称为独占锁，事务写入记录的时候获取X锁，且只允许一个事务获取X锁，其他事务需要阻塞等待。   
    S锁之间不冲突，X锁为独占锁，所以X之间会冲突，X和S也会冲突。   
    | 冲突 | S | X |
    | :-----| ----: | :----: |
    | S | 不冲突 | 冲突 |
    | X | 冲突 | 冲突 |
